# Defect Detection in Hot Rolling Mills

## Overview
This project was developed as part of the Tata Steel AI Hackathon to identify manufacturing defects in hot rolling mills using machine learning techniques.

## Problem Statement
Develop a classification model to predict defects from manufacturing process data and improve quality control.

## Dataset
- Manufacturing process dataset
- Numerical features
- Target: Defect / No Defect

## Tech Stack
- Python
- Pandas
- NumPy
- CatBoost
- Scikit-learn
- Matplotlib

## Workflow
- Data Cleaning
- Outlier Handling
- Feature Engineering
- Model Training
- Cross Validation
- Performance Evaluation

## Model
CatBoost Classifier

## Evaluation Metrics
- Accuracy
- Precision
- Recall
- F1 Score
